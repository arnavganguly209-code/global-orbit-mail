# GLOBAL ORBIT MAIL

Enterprise Mail Platform by **GLOBAL ORBIT PVT. LTD.**

Phase 1 foundation for the commercial white-label enterprise email SaaS.

## Portals

- User Portal: `webmail.theglobalorbit.com`
- Super Admin: `orbit.theglobalorbit.com`

## Quick Start

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Scripts

| Command | Purpose |
|---|---|
| `npm run dev` | Local development |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npm run lint` | ESLint |
| `npm run typecheck` | TypeScript check |
| `npm run db:generate` | Generate Prisma client |

## Stack

Next.js 15 · TypeScript · Tailwind CSS · Prisma · PostgreSQL · NextAuth-ready · TanStack Query · Framer Motion · GSAP
